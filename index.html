Postnummer.info

Letar du efter en gata och har postnummret? Behöver du kolla om en viss gata hör till ett postnummer? Vill du veta vilka gator som tillhör ett postnummer? Vill du veta vilken stad ett postnummer tillhör?
Använd detta API för att kolla. Du anger ett postnummer och får tillbaka information om vilka gator och gatnummer och vilken stad som hör till postnummret.

Du gör ett REST-anrop mot API-adressen och får en JSON-array med ett objekt för varje gata tillbaka. Vill du använda det från javascript? Inga problem, lägg på en callback= parameter och du får svaret som JSONP.

Tjänsten hämtar data från posten.se och är ett inofficiellt API. Det bör fungera så länge posten.se inte ändrar något på sin sida.

Exempel

Anrop med postnummer: 111 57

$ curl http://api.postnummer.info/v1/zip_codes/11157

      
  [
    {
      "name": "GRYTGJUTARGÅNGEN",
      "first_number": "6",
      "last_number": "6",
      "zip_code": "111 57",
      "city": "STOCKHOLM"

    {
      "name": "HÖTORGET",
      "first_number": "1",
      "last_number": "15",
      "zip_code": "111 57",
      "city": "STOCKHOLM"
    },
    ...
  ]

JSONP-anrop med postnummer: 111 57

$ curl http://api.postnummer.info/v1/zip_codes/11157?callback=parseStreets
